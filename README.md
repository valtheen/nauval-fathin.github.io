## ğŸ“ Cara Menggunakan

1. **Ganti Foto Profil**
   - Ganti file `img/profile.jpg` dengan foto profil Anda
   - Rekomendasi: 300x300px atau lebih besar, format square

2. **Update Informasi Kontak**
   - Edit `contact.html` untuk mengupdate email, phone, dan social media links
   - Update informasi lokasi jika diperlukan

3. **Tambahkan Resume**
   - Ganti `assets/resume.pdf` dengan CV/Resume Anda dalam format PDF
   - Tambahkan link download di halaman About atau Contact jika diperlukan

4. **Kustomisasi Konten**
   - Update informasi di semua halaman sesuai dengan profil Anda
   - Edit project data di `js/script.js` untuk portfolio
   - Tambahkan artikel blog di `blog.html`

5. **Deploy**
   - Upload semua file ke web hosting
   - Atau gunakan GitHub Pages, Netlify, Vercel, dll
